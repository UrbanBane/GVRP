	How to install Female Player Body Rework:

	➊ Unzip both the 'FemPedBaseModel' and 'FemPedTextures' folders.
	➋ Make sure you place these folders inside of your 'resources' folder.
	➌ Restart your server and they should be working!

	► Note: If you want to reduce the amount of folders you have, feel free to move both of the base model .ydd's into the 'PedTextures/FemaleBodyRework' folder. 
	► Note: To remove, simply delete the folders.
	► Note: Remove this README from both folders.
